##安装git
1.安装git==》sudo apt-get install git-core
2.一些全局变量的初始化
    在本地建立一个文件夹，然后做一些全局变量的初始化
    git config --global user.name 用户名或者用户ID
    git config --global user.email 你邮箱
    这两个选项会在以后你提交代码至本地仓库时自动填写到你的提交记录中去.
